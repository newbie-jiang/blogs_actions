过程

  • 参考一个demo的keil包含了哪些文件，cmake文件把demo的文件放进去就行，
  • toolchain.cmake 单独写一个编译的工具依赖，

编译命令

主目录下

sudo cmake -S . -B build -DCMAKE_TOOLCHAIN_FILE=toolchain.cmake
sudo cmake --build build

sudo rm -rf build
